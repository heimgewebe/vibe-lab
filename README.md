# 🧪 Vibe-Lab

**Das Open-Source-Labor für Vibe-Coding – Stile erforschen, Techniken erproben, Innovationen schaffen.**

> *Vibe-Coding auf ein neues Level heben: systematisch, messbar, reproduzierbar.*

---

## 🎯 Mission

Vibe-Lab ist eine kuratierte Wissensdatenbank und ein Experimentier-Labor für **Vibe-Coding** – die Kunst, mit KI-Assistenten effektiv und kreativ Software zu entwickeln. Wir sammeln, erproben, kombinieren und entwickeln Vibe-Coding-Stile, Techniken und Workflows weiter.

### Was ist Vibe-Coding?

Vibe-Coding beschreibt einen Entwicklungsansatz, bei dem der Mensch die **kreative Richtung** vorgibt und KI-Assistenten den **Code generieren**. Der Entwickler beschreibt die "Vibe" – das gewünschte Ergebnis, die Architektur, das Gefühl – und die KI setzt es um.

## 🚀 Quickstart

### Einen Katalog-Eintrag lesen

Stöbere in [`catalog/`](catalog/) – jeder Eintrag enthält:
- **Was** es ist und wie es funktioniert
- **Wann** man es einsetzen sollte
- **Synergien** mit anderen Techniken
- **Bewertung** anhand standardisierter Kriterien

### Ein Experiment starten

1. Kopiere [`experiments/_template/`](experiments/_template/) als `experiments/YYYY-MM-DD_dein-name/`
2. Formuliere deine Hypothese
3. Führe das Experiment durch und dokumentiere es
4. Erstelle einen Pull Request

### Eine Combo ausprobieren

In [`combos/`](combos/) findest du erprobte Kombinationsrezepte: Stil + Technik + Tool = maximaler Vibe.

## 📊 Bewertungsdimensionen

Jeder Ansatz wird entlang dieser Dimensionen bewertet:

| Dimension | Beschreibung |
|-----------|-------------|
| ⏱️ Geschwindigkeit | Zeit von Idee → funktionierender Code |
| 🎯 Treffsicherheit | Wie oft liefert der erste Versuch das Gewünschte? |
| 🏗️ Codequalität | Wartbarkeit, Lesbarkeit, Testabdeckung |
| 🔄 Iterationsfähigkeit | Wie gut lässt sich der Output weiterentwickeln? |
| 🧠 Kognitive Last | Wie viel muss der Mensch noch denken/korrigieren? |
| 📐 Skalierbarkeit | Funktioniert es auch bei größeren Projekten? |
| 🎨 Kreativität | Liefert der Ansatz innovative Lösungen? |

## 🏷️ Vibe-Maturity-Model

Wo stehst du auf deiner Vibe-Coding-Reise?

| Level | Name | Beschreibung |
|-------|------|-------------|
| 1 | **Ad-hoc** | Zufälliges Prompting, Trial & Error |
| 2 | **Bewusst** | Gezielte Technikwahl, erste Muster erkannt |
| 3 | **Systematisch** | Reproduzierbare Workflows, konsistente Ergebnisse |
| 4 | **Optimiert** | Datengetriebene Stilwahl, Benchmarks nutzen |
| 5 | **Innovativ** | Eigene Stile entwickeln, Meta-Optimierung |

## 💡 Innovative Konzepte

### 🧬 Vibe-DNA
Jeder Vibe-Coding-Stil als "Gen" – kombinierbar zu neuen Hybrid-Stilen. Visualisierung als interaktive Kombinationsmatrix.

### 🤖 Meta-Vibe-Coding
KI, die den optimalen Vibe-Coding-Stil für ein gegebenes Problem empfiehlt. Self-Improving durch Ergebnis-Feedback.

### 🎮 Vibe-Challenges
Standardisierte Coding-Aufgaben, gelöst mit verschiedenen Stilen. Vergleichbare Ergebnisse zeigen: Welcher Stil passt zu welchem Problem?

### 🔬 Prompt-Forensik
Reverse-Engineering erfolgreicher Vibe-Sessions. Was genau im Prompt hat den Unterschied gemacht?

### 🧩 Vibe-Coding-Periodensystem
Visuelle Darstellung aller Techniken – gruppiert nach Eigenschaften, interaktiv erkundbar.
