# 🎮 Challenge 2: Frontend-Komponente mit State Management

## Beschreibung

Baue eine **interaktive Kanban-Board-Komponente** mit Drag & Drop – als React oder Vue Komponente.

## Anforderungen

### Funktional
- Kanban-Board mit mindestens 3 Spalten (Todo, In Progress, Done)
- Karten erstellen, bearbeiten und löschen
- Drag & Drop zwischen Spalten
- Persistenz im LocalStorage
- Responsive Design (Mobile + Desktop)

### Technisch
- React oder Vue mit TypeScript
- Kein UI-Framework für das Board selbst (Tailwind/CSS erlaubt)
- State Management (Zustand, Pinia, oder Context)
- Mindestens 3 Komponenten-Tests

### Nice-to-Have (optional)
- Dark Mode
- Keyboard-Navigation
- Animationen

## Bewertungskriterien

| Kriterium | Gewicht |
|-----------|---------|
| Funktioniert Drag & Drop? | 25% |
| UI/UX Qualität | 25% |
| Codequalität und Komponentenstruktur | 25% |
| Responsive Design | 15% |
| Tests | 10% |

## Zeitmessung

Stoppe die Zeit von der ersten Interaktion bis zum funktionierenden Kanban-Board.
