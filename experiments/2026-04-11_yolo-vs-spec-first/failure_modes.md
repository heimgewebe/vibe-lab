# Failure Modes

- **Überstrukturierung (Spec-First)**: Zu viel Zeit wird auf Spezifikationen verwandt, wodurch der Geschwindigkeitsvorteil des KI-Tools verloren geht.
- **Kontextverlust (YOLO)**: Bei komplexeren Projekten führt der YOLO-Ansatz schnell zu einem Punkt, an dem das Modell den Kontext verliert und inkonsistenten Code generiert.
